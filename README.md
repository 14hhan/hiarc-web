# hiarc-web

Gatsby와 Netlify를 이용한 HI-ARC의 새 웹페이지

## 시작하기

npm, Node.js, Gatsby가 설치되어있지 않다면 먼저 설치하여야 합니다.

1. 이 저장소를 fork, clone 합니다.
2. 터미널에서 다음의 명령어를 입력합니다.

   ```zsh
   # hiarc-web 폴더로 이동
   > cd hiarc-web

   # 의존성 설치
   > npm install

   # 로컬환경에서 개발 모드 실행
   > npm run develop
   ```

3. 웹브라우저를 이용하여 `https://localhost:8000`으로 접속합니다.
